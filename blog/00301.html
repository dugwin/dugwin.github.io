Зимнее время?

Пару лет назад, когда я писал движок для этого блога (кстати, никому аналогичный не нужен? отдам даром) я писал функцию для пересчёта времени UTC в московское, с учётом перевода на зимнее время, выглядело это так:

if d.year >= 2015:

return d + datetime.timedelta(hours = 4) # переход на зимнее время будет отменён

if d.year >= 2024:

return d # все будут жить по UTC

Когда писал этот код, никакой речи об отмене летнего/зимнего времени вообще не шло. Оказалось, чуть-чуть совсем ошибся.
